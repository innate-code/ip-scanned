Бывают ли такие моменты, что нужно пробить по ip? Если да, то к твоему вниманию утилита которая позволит это осуществить.

Установка

Обновляем пакеты:
apt update && apt upgrade

Устанавливаем зависимости:
apt install git python
pip install request

Качаем сам инструмент:
git clone https://github.com/tenych69/ip-ban
cd ip-ban

Запуск
python ip-ban.py

Далее надо выбрать один из пунктов:
(1) - Информация по айпи
(2) - Мой айпи
(0) Выход

В первом пункте указываем айпи, и тут же получаем кучу информации
